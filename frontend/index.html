Como configurar o Google Meet:

  1. Acesse console.cloud.google.com
  2. Crie um projeto e ative a Google Calendar API
  3. Em "Credenciais" → Criar → OAuth 2.0 → Aplicativo de Desktop
  4. Baixe o JSON e salve como backend/credentials.json
  5. No terminal do backend, execute: python setup_google_auth.py
  6. Uma aba do navegador abrirá para autorizar. Clique em Permitir.

Como configurar no Meta for Developers:

  1. Acesse developers.facebook.com → seu App → Webhooks
  2. Adicione um webhook para o objeto Page (ou leadgen)
  3. Cole a URL acima em "URL de Callback" e o Token de Verificação no campo correspondente
  4. Clique em Verificar e Salvar — o backend responderá ao desafio automaticamente
  5. Assine o campo leadgen na sua Página
  6. Gere um Page Access Token de longa duração e cole acima
  7. Salve as configurações aqui e pronto ✓
