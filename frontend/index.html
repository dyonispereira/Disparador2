Como configurar o Google Meet:

  1. Acesse console.cloud.google.com
  2. Crie um projeto e ative a Google Calendar API
  3. Em "Credenciais" → Criar → OAuth 2.0 → Aplicativo de Desktop
  4. Baixe o JSON e salve como backend/credentials.json
  5. No terminal do backend, execute: python setup_google_auth.py
  6. Uma aba do navegador abrirá para autorizar. Clique em Permitir.
